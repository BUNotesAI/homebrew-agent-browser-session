# homebrew-agent-browser-session

Homebrew tap for [agent-browser-session](https://github.com/BUNotesAI/agent-browser-session) — headless browser automation CLI for AI agents.

## Install

```bash
brew install BUNotesAI/agent-browser-session/agent-browser-session
```

Or tap first:

```bash
brew tap BUNotesAI/agent-browser-session
brew install agent-browser-session
```

## Requirements

- Node.js (installed automatically as a dependency)
- Chromium browser (installed automatically via `patchright` on first run)

## Update

```bash
brew upgrade agent-browser-session
```
